# Boilerplate NestJS Docker
## Usage example
### Step 1: Init project
* Build docker
```
docker-compose build
```
* Start docker compose
```
docker-compose up -d
```

### Step 2: Access
* Rest full api on same 3005 port:
  <a href="http://localhost:3005/api/v1">localhost:3005/api/v1</a>
  <br/>

## Other
### Log System

* api-server
```
docker logs --tail 1000 api-service 
```
### Connect database use DataGrip
* use dataGrip
```
Host: localhost
Port: 27017
User: root
password: root
Database: service
URL: mongodb://xxxx:xxx/xxx?authSource=admin
```
